# tradingchain-sdk

在项目描述文件里添加gitpack和依赖

#### maven
```
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>com.github.xuqplus2</groupId>
        <artifactId>java-sdk-test</artifactId>
        <version>0.0.1</version>
    </dependency>
</dependencies>
```
